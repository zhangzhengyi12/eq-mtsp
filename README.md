## 名词概念

**gens**

基因，最小单元，被映射成地图中的点

**chromesome 染色体**

用基因组成，被映射成地图中的一个旅行商所走的路径

**representation 基因组**

由多个 chromesome 染色体组成，被映射成最终解决方案
